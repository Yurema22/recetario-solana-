# Recetario Descentralizado en Solana

## Descripción

Este proyecto consiste en un **recetario descentralizado** desarrollado en la blockchain de Solana. Permite almacenar y administrar recetas mediante operaciones CRUD (crear, leer, actualizar y eliminar).

El programa fue desarrollado utilizando **Rust** con el framework **Anchor**, y se utiliza **TypeScript** para interactuar con el programa y realizar pruebas.

## Funcionalidades

* Crear recetario
* Agregar recetas
* Ver recetas almacenadas
* Cambiar disponibilidad de una receta
* Eliminar recetas

## Estructura de una receta

Cada receta contiene:

* Nombre
* Ingredientes
* Preparación
* Disponibilidad

## Ejecución

Compilar el programa:

```
anchor build
```

Ejecutar pruebas:

```
anchor test
```



Proyecto académico desarrollado para aprender el funcionamiento de programas en la blockchain de Solana.
